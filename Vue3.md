## CSS

### css 选择器修饰符

#### :solted()

选择器只作用于插槽里面的内容

```vue
<style scoped>
:solted(div) {
    color: red;
}
</style>
```

#### :deep()

组件库样式穿透

```vue
<style scoped>
:deep(div) {
    color: red;
}
</style>
```

组件库样式穿透

#### :global()

全局生效

```vue
<style scoped>
:global(div) {
    color: red;
}
</style>
```

#### 动态 CSS

```vue
<template>
    <div class="div">动态CSS</div>
</template>

<script setup lang="ts">
import { ref } from "vue";
const style = ref("red");
</script>

<style scoped>
.div {
    color: v-bind(style);
}
</style>
```

## 项目应用

#### 动态引用静态资源

```typescript
const getAssetsFile = (url: string) => {

};
```

#### 自定义组件属性

##### 定义全局属性

```typescript
// main.ts
app.config.globalProperties.$dateFormater = dateFormater;
app.config.globalProperties.$getDate = getDate;
```

##### ts 文件类型声明

```typescript
// customProperties.d.ts
export declare module "vue" {
    interface ComponentCustomProperties {
        $dateFormater: (time: string | any, pattern: string) => string;
        $getDate: (dateStr: string) => {
            date: string;
            imageIndex: number;
        };
    }
}
```
